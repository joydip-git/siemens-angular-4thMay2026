1. Binding:
	a. Data Binding: technique to bind (pass) a data from component code to the same component template
	syntax: {{prop-name}} -> interpolation
		<element [attr]="prop-name"> -> property binding
	note: property binding does not stringify the property value whereas the interpolation does so.
	b. event binding: technique to pass data from component template to the same component code
	syntax: <element (event)="method-in-code([args])">
