Form Handling
------------------------
1. template driven form
	controlling form data handling from the template of the form, rather than code
	in this aspect you use several attribute directives to achieve that
	ngForm => creates formgroup object and binds to the <form> element and suppresses the behavior of form server side submission
	ngModel => creates formcontrol object and binds to the <input> element
2. reactive form

attribute directive:
----------------------------
ngModel

@Directive({
	selector:'[ngModel]',
	standalone:false
})
export class NgModelDirective{
}

module name
--------------------
FormsModule

@NgModule({
	//register components, directives and pipes
	declarations:[NgModelDirective],

	//register services here
	providers:[]

	//import modules (feature and built-in)
	imports:[]

	//name of the compoments, pipes, directives and other modules which can be exported to other modules
	exports:[NgModelDirective]
})
export class FormsModule{
}

package:
-----------
@angular/forms
